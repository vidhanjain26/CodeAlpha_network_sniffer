# Network Sniffer (Python GUI)

This project is a Python-based GUI network packet sniffer using Scapy and Tkinter. It captures live network traffic and displays source/destination IPs, protocols, ports, and payload data in real time.

## Features
- Real-time packet capturing
- GUI interface with Start/Stop buttons
- Displays IP addresses, protocols, ports, and payload
- Works on Windows using Npcap

## Requirements
- Python 3.x
- Scapy
- Npcap (Windows)
